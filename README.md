# *Dream Car Rental*

## *Description*
*Dream Car Rental* is a professional car rental platform designed to provide users with a seamless experience for renting both *luxury* and *standard vehicles. This project is built using **HTML and CSS* to ensure a high-quality, user-friendly interface. Beyond web development, it incorporates *Microsoft Excel* for comprehensive data management, including tracking car availability, rental pricing, and *monthly income analysis. The project aims to **digitalize* rental operations and leverage data for better business decision-making.
